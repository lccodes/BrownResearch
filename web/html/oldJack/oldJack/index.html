JACK

Java Auction Configuration Kit

What is JACK?

JACK is a simulator of general auction games. JACK lets you:
  • configure simple (1st, 2nd, ascending, descending) auction games
  • combine these simple auctions into more complex auction schedules (sequential and/or simultaneous)
  • pick between different parametrizable valuation functions
  • control the prior over which these valuation functions are drawn
