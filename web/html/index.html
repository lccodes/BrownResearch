JACK:

Java Auction Configuration Kit

What is JACK?

JACK is an in-the-box configurable auction simulator designed to allow researchers to rapidly create and test different auction configurations.

JACK lets you:

  • configure simple (1st, 2nd, ascending, descending) auction games
  • combine these simple auctions into more complex schedules (sequential and/or simultaneous) of auctions
  • pick between 5 different value functions
  • control the prior over which these valuation functions are drawn
